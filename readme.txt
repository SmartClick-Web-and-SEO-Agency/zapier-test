# SmartClick HTML/SCSS Starter

## Basic Features

- Comes with Bootstrap (v4) Sass source files and additional .scss files. Nicely sorted and ready to add your own variables and customize the Bootstrap variables.
- Uses a single minified CSS file for all the basic stuff.
- [Font Awesome](http://fortawesome.github.io/Font-Awesome/) integration (v4.7.0)

## Sass Structure
In assets/scss are all the folders for the structure. If you want to change some variables go to abstracts/theme-variables.scss

Read this link to understand our structure:https://www.webdesignerdepot.com/2020/12/2-smartest-ways-to-structure-sass/. Try using as much as possible of the CSS utilities. Create new Utilities classes in the Abstract folder.


## Instalation
### Installing Dependencies
- Make sure you have installed Node.js and Browser-Sync (optional) on your computer globally
- Then open your terminal and browse to the location of the git repo
- Run: `$ npm install`

### Running
To work with and compile your Sass files on the fly start:

- `$ npm run watch`
